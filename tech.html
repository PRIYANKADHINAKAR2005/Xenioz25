Code Gamble is a strategic programming event where teams must bid on the time they require to solve a given coding problem. The lower the bid, the earlier they get a chance to solve it.








Aspiring wizards will showcase their magical discoveries on cutting-edge enchantments before a council of esteemed professors and judges.






Participants will debug multiple code snippets within a given time. A row of systems will be set up, each with a different Java or C++ question. Participants must debug the code within the allocated time before moving to the next system.


Round 1: MCQ Challenge (Points Accumulation)
  1. Each team gets an MCQ with 30 seconds to answer.
  2. Scoring:
    • Correct: 500 points
    • Wrong/No Answer: Passed to next team for 250 points
Round 2: Bidding Round (Point Boosting)
  1. All teams get the same coding question & bid their solving time.
  2. Scoring:
    • Solved in bid time: (Max Bid - Your Bid) Ã— 100 points
    • Failed: 0 points
Round 3: Final Bidding (Elimination Round)
  1. Top 5 teams qualify.
  2. Two coding problems, teams bid for solving time.
  3. Lowest bidder attempts first; failure passes the chance.
  4. Scoring:
    • Solved: Points doubled
    • Failed: Lose 50% of points
  5. Top 3 teams win!
Domains:

-Artificial Intelligence & Machine Learning
- Cybersecurity & Blockchain
- Cloud Computing & Big Data
- Internet of Things (IoT)
- Software Engineering & Development
- Computer Vision & Image Processing



Time Allotment:

Presentation: 10 minutes
Q&A Session: 5 minutes
Total Duration: 1.5 hours (depending on the number of participants)


Event Format:

  1. Individual event.
  2. Questions will be given in Java or C++.
  3. Time Limit: 2 minutes per system.

Rules:
  1. Movement: After 2 minutes, participants must rotate to the next system.
  2. No Return Rule: Participants cannot return to previously attempted systems.
  3. Winning Criteria: The participant who debugs the most problems first wins. In case of a tie, completion time will be considered.
  4. Disqualification: Copying code from another participant or tampering with code structure is prohibited.
